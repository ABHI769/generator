# CertiGen — Bulk Certificate Generator

Generate individual downloadable certificates for multiple students from a single template.

## How to use

1. Open `index.html` in any modern browser (Chrome, Edge, Firefox).
2. **Template**: Upload your own certificate image (PNG/JPG), or click **Use default template**.
3. **Names**: Enter one student name per line in the text box.
4. **Placement**: Adjust font, size, color, and vertical position until the live preview looks right.
5. Click **Generate Certificates** — each name gets its own certificate.
6. Download certificates individually, or use **Download All (ZIP)**.

## Custom templates

Upload a certificate image with a blank area where names should appear. Use the **Vertical position** slider to align the name on your template.

All processing runs locally in your browser — no server or upload required.

## Files

- `index.html` — page structure
- `styles.css` — layout and styling
- `app.js` — certificate rendering and downloads
